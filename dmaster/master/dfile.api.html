dFile Store API

Documentation

To do a request to the dFile Store API, you have to perform an HTTP request, in the following form http://username:password@dfile.ekt.gr/.
The available methods, generally are SubmitOJS, SubmitPhDMS, CreatePdf.
Each one of these methods should be implemented using GET or POST HTTP methods, as indicated below.

Method Request Method
SubmitOJS POST
SubmitPhDMS POST
CreatePdf POST
